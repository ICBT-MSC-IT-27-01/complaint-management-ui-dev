# Complaint Management System - UI

## Overview
This repository contains the Angular frontend application for the Complaint Management System.  
It provides a responsive user interface for managing complaints, users, and system workflows.

## Tech Stack
- Angular
- TypeScript
- Bootstrap / CSS
- REST API Integration

## Features
- User authentication & authorization
- Create, update, and track complaints
- Dashboard with complaint statistics
- Role-based access control
- Responsive UI design

## Prerequisites
- Node.js (v18 or later recommended)
- Angular CLI

## Installation

```bash
npm install
